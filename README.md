# Dotfiles

Cross-platform dev configs managed with [chezmoi](https://chezmoi.io/).

## Quick Start

```bash
chezmoi init https://github.com/trinhlehainam/Dotfiles.git
chezmoi diff
chezmoi apply
```

## Where To Edit

- Neovim: `home/.shared-configs/nvim/`
- WezTerm: `home/dot_config/wezterm/`
- Windows configs: `home/AppData/`

📁 `dot_config/` → `~/.config/`

## How Neovim Config is Managed

**Edit source:** `home/.shared-configs/nvim/`

**Chezmoi hook config:**
- Bootstrap template: `home/.chezmoi.toml.tmpl`
- Managed active config: `home/dot_config/chezmoi/chezmoi.toml.tmpl`

**Auto-generated by hook** (do not edit):
- Unix/Linux/macOS: `home/dot_config/nvim/**/*.tmpl`
- Windows: `home/AppData/Local/nvim/**/*.tmpl`

Changes to generated wrapper files are overwritten.

## Layout

```text
home/
  .shared-configs/nvim/     # Neovim source
  dot_config/               # Unix app configs (chezmoi-managed)
  AppData/                  # Windows app configs (chezmoi-managed)
```

## Requirements

- chezmoi + git
- A package manager (brew/winget/apt/…)
