---Set up Kali linux image with Windows 10:
	- Prepare USB for boot ( capacity > 4.0G )
	- Install Rufus version close to 3.0.0 ( which has DD mode )
	- Download Kali linux Live version
	- Burn ISO to USB with Rufus
	
---Install boot:
	- Restart computer and enter to BIOS mode
	- Select USB drive
	- Install
