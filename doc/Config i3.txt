- type i3-config-wizard in terminal if doesn't have i3 config
- Install: playerctl (media controll in bash)
- Install: feh (app for show background image)
- Install arandr (UI for xrandr, manage display monitor) 
- Use xprop to identify application's window class (use to identify class name to able to assign to workplace)
- Custom and assign app to workplace  
- Install xlappearance to manage gtk (UI theme config) (windows, folder, icon, ...)
- Move all ttf file to $HOME/.fonts
- Install: custom theme (Arc, Papirus, Moka, ...)
- Install: rofi (dmenu replacement)
- Install: compton to support transparency for i3
- Install: i3blocks to custom i3 status bar
