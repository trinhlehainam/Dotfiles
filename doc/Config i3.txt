- type i3-config-wizard in terminal if doesn't have i3 config
- Install: playerctl (media controll in bash)
- Install: feh (app for show background image)
- Install: arandr (UI for xrandr, manage display monitor) to create shell script for xrandr 
    + Use --scale to scale monitor 
- Use xprop to identify application's window class (use to identify class name to able to assign to workplace)
- Custom and assign app to workplace  
- Install lxappearance to manage gtk (UI theme config) (windows, folder, icon, ...)
- Move all ttf file to $HOME/.fonts
- Install: custom theme (Arc, Papirus, Moka, ...)
- Install: rofi (dmenu replacement)
- Install: picom (because compton is no longer mantained) to support transparency for i3
- Install: i3blocks to custom i3 status bar
- Install: pavucontrol to check volume in status bar match system volume
- insert this line to [volume] block in i3blocks config to update volume in status bar:
    + command=/usr/share/i3blocks/volume 5 pulse
